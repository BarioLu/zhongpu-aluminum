A modern, responsive website for Zhongpu Aluminum, a professional decorative aluminum profile manufacturer from Foshan, China.

### Home Page
- **Hero Banner**: Eye-catching banner with company slogan and quick inquiry form
- **Company Advantages**: 5+ years experience, OEM & ODM, Export to 30+ countries, ISO Certified
- **Hot Products**: Featured aluminum profile products with applications and features
- **Surface Finishing**: Overview of available surface treatment options
- **Call-to-Action**: WhatsApp integration and contact forms

- **Production Capability**: 12 extrusion lines, 2 anodizing lines, 3 powder coating lines

## 🚀 Getting Started

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd zhongpu-aluminum-site
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:3000`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
